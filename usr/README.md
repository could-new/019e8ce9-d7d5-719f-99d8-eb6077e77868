# Kenneth & Zerah's Universe

A private romantic universe that preserves memories, milestones, and emotional moments. It feels like a calm, soft, starry universe where each memory is a "star" and every section feels personal and warm.

## Features
- **Starry Background**: A gentle animated space background.
- **Home Screen**: A glowing central planet with a relationship counter.
- **Memory Timeline**: A timeline of relationship memories.
- **Photo Gallery**: Star clusters of uploaded photos.
- **Shared Journal**: A private couple diary.
- **Open When Messages**: Fill-in-the-blank messages for different moments.
- **Music Playlist**: Shared songs.

## Setup
1. Run `flutter pub get`
2. Run `flutter run`
